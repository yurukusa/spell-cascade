# Kenney Asset Packs for Spell Cascade

All assets are CC0 licensed from kenney.nl - free to use for any purpose without attribution.

## Downloaded Packs

### 1. tiny-dungeon/ (136 PNG files)
- 16x16 pixel tiles
- Top-down dungeon tileset
- Characters, enemies, items, floor/wall tiles
- Perfect for: player character, enemies, dungeon environment

### 2. top-down-shooter/ (587 PNG files)
- Character sprites with animations (gun, hold, reload, etc.)
- Multiple character types: soldiers, zombies, robots
- Weapon sprites
- Tile assets
- Perfect for: player character variations, enemy types, weapons

### 3. pixel-shmup/ (150 PNG files)
- Pixel art shoot-em-up assets
- Ships and aircraft
- Projectiles and bullets
- Effects and explosions
- Perfect for: projectile sprites, bullet patterns, spell effects

### 4. particle-pack/ (193 PNG files - 512x512 high-res)
- Fire, flames, smoke effects
- Magic effects (magic_01-05.png)
- Circles, flares, lights
- Muzzle flashes
- Sparks and electricity
- Available in transparent and black background versions
- Perfect for: spell effects, particle systems, visual feedback

### 5. roguelike-rpg-pack/ (Spritesheet - 1700+ tiles)
- 16x16 pixel tiles
- Comprehensive roguelike tileset
- Characters, monsters, items, environments
- Available as spritesheet (needs extraction)
- Perfect for: enemies, items, environmental variety

### 6. micro-roguelike/ (326 PNG files)
- Micro/tiny pixel art roguelike assets
- Monochrome and colored versions
- Characters, enemies, items
- Perfect for: minimalist aesthetic, UI icons, small sprites

### 7. pixel-ui-pack/ (36 PNG files)
- UI elements and frames
- 9-slice panels for scalable UI
- Buttons, borders, containers
- Multiple style variants (Ancient, Colored, Outline)
- Perfect for: health bars, XP bars, menus, HUD elements

## Usage Recommendations for Spell Cascade

- **Player Character**: tiny-dungeon tiles or top-down-shooter characters
- **Enemies**: tiny-dungeon monsters, top-down-shooter zombies/robots, roguelike-rpg-pack creatures
- **Projectiles/Bullets**: pixel-shmup projectiles, custom colored circles
- **Spell Effects**: particle-pack magic effects, flames, circles with glow
- **XP Orbs**: particle-pack circles/lights with different colors
- **Health Bars**: pixel-ui-pack 9-slice panels
- **Visual Effects**: particle-pack sparks, flares, muzzle flashes for impact

## File Structure

Each pack maintains its original structure:
- Individual PNG files in Tiles/ or PNG/ directories
- Spritesheets where applicable
- Vector sources in some packs
- Unity samples in particle-pack
